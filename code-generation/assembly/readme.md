this is a readme for hack machine assemble language

HACK machine have three main operations, in short:

1- assign A register, with the syntax @xxx, where xxx could be a contant or variable
2- assign values to (M, A, or D registers), with the syntax M = M + 1, D = M - D, etc
3- jump to specefic location, with the syntax 0; JMP, where the jump will be to the location specefied by A register

in general the syntax will be: DEST = COMP; JUMP
where COMP is mand, DEST and JUMP is optional
